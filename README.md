Este repositorio está dividido en dos módulos principales: **Backend** y **Frontend**.  
Cada uno funciona de manera **independiente** y cuenta con su **propio README** con instrucciones específicas de configuración, ejecución y despliegue.

---


## Módulos

### Backend
Contiene toda la lógica del servidor, APIs, base de datos, reglas de negocio y servicios.

Incluye:
- Configuración del entorno
- Variables de entorno
- Scripts de ejecución
- Comandos de build
- Despliegue
- Arquitectura del backend

Consultar:
```

/backend/README.md

```

---

### Frontend
Contiene la aplicación cliente (interfaz de usuario, vistas, estado, integraciones, etc).

Incluye:
- Instalación de dependencias
- Configuración del entorno
- Comandos de desarrollo
- Build de producción
- Despliegue

Consultar:
```

/frontend/README.md

````
